# project

Project create with [whatsup cli](https://github.com/whatsup/cli)

## Usage

```sh
# move to create folder
$ cd [PROJECT_NAME]

# install dependencies
$ npm install

# serve with hot reload
$ npm run start

# build project for production
$ npm run build
```
